# CLI 会话捕获

真实执行命令，原样记录输出。不设占位符，不搞重放校验。尽量使用真实命令，而不是类似dist/index.js xxx -- yy 这样的开发命令。

## 存放位置

`.apm/kb/docs/iterations/{current}/test/<主题>.md`

`{current}` 从当前迭代目录名获取，无法确定时询问用户。

## 文档模板

```markdown
# CLI 验收：<标题>

- 日期: <YYYY-MM-DD>
- 审查人: pending

## <场景名>

```bash
<实际执行的命令>
```

标准输出:
```
<原样粘贴>
```

标准错误:(不存在则隐藏)
```
<原样粘贴>
```
---(分隔符拆分多条命令)

备注: <可选，比如临时目录路径>
```

## 要求

1. 必须真实执行，禁止编造输出
2. 在临时目录中运行，但直接把实际路径写进备注，不用占位符
3. 多个场景顺序执行，状态延续
```
